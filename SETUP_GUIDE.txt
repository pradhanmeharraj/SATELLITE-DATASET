# ============================================================================
# COMMON ISSUES & SOLUTIONS
# ============================================================================

Issue: CUDA out of memory
Solution: 
  - Reduce batch size: --batch-size 16
  - Reduce model size: --backbone resnet18
  - Use CPU: --device cpu

Issue: Dataset not found
Solution:
  - Use dummy data: --dummy-data
  - Download Kaggle dataset to data/ folder
  - Check data directory structure

Issue: Model not improving
Solution:
  - Increase learning rate: --learning-rate 5e-3
  - Train longer: --num-epochs 200
  - Try different backbone: --backbone resnet101
  - Check data augmentation in config.py

Issue: Import errors
Solution:
  - Reinstall requirements: pip install -r requirements.txt --upgrade
  - Check Python path in scripts
  - Use virtual environment: python -m venv venv

Issue: Slow training
Solution:
  - Increase batch size if memory allows
  - Use fewer workers: --num-workers 2
  - Use smaller model: --backbone resnet18
  - Enable GPU: --device cuda
